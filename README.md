The compose file is for my specific set/directory on my server

Within the folder used you need:
1) keys.txt <- line separated gemini API keys
2) discordtoken.txt <- your discord bot token
3) admins.txt <- your list of admin IDs (right click -> copy user ID with developer mode on)

then just add the bot to your server with the "Read message history", "view messages" and "send messages" oath2 permissions and discord server permissions when you paste the url in your browser from the discord bot page
